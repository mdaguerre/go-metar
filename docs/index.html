View on GitHub
Install
Homebrew $ brew tap mdaguerre/tap && brew install go-metar
Go $ go install github.com/mdaguerre/go-metar@latest
Usage
zsh
$ go-metar KJFK # decoded weather report
$ go-metar EGLL --raw # raw METAR string
$ go-metar LFPG --all # both formats
$
