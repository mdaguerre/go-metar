GO-METAR

Aviation Weather from Your Terminal

Real-time METAR Data
Installation
Homebrew (macOS/Linux) Recommended
$ brew tap mdaguerre/tap && brew install go-metar
Go
$ go install github.com/mdaguerre/go-metar@latest
Usage
go-metar KJFK Decoded weather report
go-metar EGLL --raw Raw METAR string only
go-metar LFPG --all Both raw and decoded
